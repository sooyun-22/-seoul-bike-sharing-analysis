## 📌 프로젝트 개요

서울에서 직접 매일 보이는 따릉이. 출근·여가·심야 이동 패턴이 실제 데이터에서도 어떻게 나타나는지 궁금했습니다.
단순한 이용량 파악을 넘어, **시간·이용자·날씨·공간을 종합 분석**하여 도시 정책에 활용 가능한 실용적인 인사이트를 도출하는 것을 목표로 했습니다.

## 📂 데이터

| 데이터 | 출처 | 내용 |
|--------|------|------|
| 서울 자전거 공유 데이터 | Kaggle | 시간별 대여 기록 + 기상 관측 데이터 |
| 서울시 공공자전거 대여이력 (9월, 10월) | 서울 열린 데이터 광장 | 이용자별 개별 대여 기록 |
| 서울시 공공자전거 대여소 정보 | 서울 열린 데이터 광장 | 대여소 위치·ID 등 공간 메타데이터 |

> 전체 대여이력 데이터 중 0.5% 랜덤 샘플링하여 분석에 활용

---

## 🔍 분석 흐름

```
데이터 수집 → 전처리 → 데이터 결합 → 시각화 및 분석 → 활용 방안 도출
```

1. **전처리** : 결측값 제거, 열 이름 통일, 불필요한 열 삭제, rain_flag 파생변수 생성
2. **데이터 결합** : 대여이력 + 날씨 + 대여소 정보를 datehour 기준으로 병합
3. **시간대 분석** : 전체·평일/주말·성별·연령대별 시간대 패턴
4. **이용자 특성 분석** : 출근형 vs 여가형 이용 비중, 연령대별 평균 이용 시간
5. **날씨 영향 분석** : 기온·습도·풍속·강수량이 대여량에 미치는 영향, 쾌적지수 분석
6. **공간 분석** : 대여소 위치 기반 히트맵, 시간대·요일별 공간적 패턴 차이
7. **활용 방안** : 대여소 재배치, 자전거 재배치 전략, 날씨 기반 수요 예측 모델 제안

---

## 📊 주요 분석 결과

**시간대 패턴**
- 출퇴근 시간(7~10시, 17~19시)에 뚜렷한 이중 피크
- 평일은 출퇴근 집중, 주말은 낮 시간대 여가 중심 패턴

**이용자 특성**
- 남녀 모두 출근형 이용 비중이 약 65%로 여가형의 2배
- 30대가 출퇴근 패턴이 가장 선명, 20대는 저녁 피크가 가장 강함

**날씨 영향**
- 비 오는 날 대여량이 평시의 약 1/3 수준으로 급감
- 쾌적지수 0.4~0.6 구간에서 대여량 최대 → 너무 더운 날도 감소
- 습도가 높아질수록 대여량 감소 경향 뚜렷

**공간 분석**
- 마곡나루·망원·잠실·건대 축에서 고밀도 집중
- 평일: 업무지구 중심 / 주말: 한강 축 여가 지역 중심으로 패턴 전환
- 여가형 대여소가 날씨 민감도 가장 높고, 업무지구는 비 와도 수요 유지

---

## 💡 활용 방안

- **대여소 재배치** : 시간대·요일별 수요 패턴으로 고수요 지역 대여소 추가 배치
- **자전거 재배치 전략** : 출퇴근 시간대 수요 급증 예측 → 사전 배치 자동화
- **날씨 기반 수요 예측 모델** : 기온·습도·강수량 조합으로 시간대별 예상 수요 도출
- **지역 특성 기반 운영 전략** : 업무지구 vs 여가지역에 서로 다른 운영 전략 적용
